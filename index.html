Introduction

Valanalysis is a fan-based website dedicated to helping newcomers to the game learn all about the basics. Discover all of the agents you can play, the history of the game, and some useful tips and tricks to get your gameplay competitive. Join a fanbase that is over 18 million players strong, and become the best!

What is Valorant?

Valorant is a tactical shooter that was created by Riot Games in 2020. This game is set up as a 5v5 scenerio with both attackers and defenders. There are many different agents that you can choose from, and each agent has unique abilities that sets them apart from the others. Choose your agents that work well together and achieve victory.
